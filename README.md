# 📚 Estudos Java IDE

Projeto educacional com exercícios e exemplos de Java para iniciantes.

## 📁 Estrutura

```
src/aulas/
├── licoesdeaula/        # Aulas teóricas
│   ├── aula10/          # Variáveis
│   ├── aula11/          # Tipos de dados
│   ├── aula12/          # Entrada (Scanner)
│   ├── aula13/          # Operadores
│   └── aula14/          # if/else, switch
└── licoesdecasa/        # Exercícios práticos
    ├── aula13/          # ~16 exercícios
    └── aula14_15/       # ~20 exercícios
```

## 🎯 Tópicos

- Variáveis e tipos de dados
- Operadores
- Estruturas de controle
- Leitura de dados

## 🚀 Como usar

1. Abra em uma IDE Java (IntelliJ, Eclipse, NetBeans)
2. Execute o arquivo desejado
3. Pratique os exercícios

## 🛠️ Requisitos

- Java JDK 8+
